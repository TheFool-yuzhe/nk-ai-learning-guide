## 写在前面

上海交通大学《生存手册》中曾写道：

> "国内绝大部分大学的本科教学，不是濒临崩溃，而是早已崩溃。"

依笔者之见，国内的研究生教育也大抵如此——即使你上了研究生，也几乎不会有什么课程对你的就业产生实质性帮助。

如果你有幸进入本校本院，就会发现大学前一两年的时光和高中其实没有本质区别，尤其在评价体系上：上课、做题、考试。富有特色的月考制度，更是让你恍惚间以为还在读高中。

但很遗憾，这种日子过不了多久。除了极少数选择数学/统计学术道路的同学（直博），大多数人在本科三年级开始，就突然被要求去"拥抱 AI 时代"了——尽管你们对 AI 可能还是一无所知。大家转向 AI 的原因有很多，但很重要的一条无非是"算法给钱多"；当然，身处 AI 最前沿，至少也不至于立马被淘汰。

**本教程的目标，就是帮你用最快的速度，从对 AI 一无所知，到逐渐走上这条道路。**

## 使用指南

AI 行业更新极快，涉及面也非常广。本教程只求尽可能带你入门，帮你了解当下最热门的方向并从中切入。本教程尽可能简单易上手，你可以在一个下午的时间内从 0 开始学习 AI。因此：

- 如果本教程超过 **半年** 未更新，其参考意义将大打折扣——届时你可以直接联系仓库作者寻求帮助。
- 本教程的前置要求：
  - **不超过本科一年级的数分/高代（或高数/线代）基础**，否则在涉及部分公式时会有困难。
  - **基础的 Python 代码编写能力**（特别是对列表、元组、字典等数据格式的了解）。
  - 其余内容均可零基础上手。

## 内容

在开始学习之前，想先给大家打个气：据我了解，国内计算机专业的本科生，在本科低年级（四年级之前）也几乎没有接触过大模型相关的基础课程。所以，如果你是数学/统计或其他非计算机专业出身，不必觉得自己和计算机的同学差距很大——本文涵盖的最新技术，他们同样没学过。

请按照 [学习教程](tutorial.md) 进行学习。
